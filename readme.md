# YaCut — URL Shortener

Сервис сокращения ссылок с поддержкой пользовательских алиасов и REST API.

## Описание проекта

Проект решает задачу упрощения работы с длинными URL.

Позволяет:
- создавать короткие ссылки для удобного обмена
- использовать пользовательские алиасы
- автоматически генерировать уникальные короткие идентификаторы
- выполнять редирект на оригинальный URL

## Что реализовал

- Разработал веб-приложение на Flask
- Реализовал REST API для работы с короткими ссылками
- Спроектировал модели данных (ссылки, алиасы)
- Реализовал бизнес-логику:
  - генерация уникальных коротких идентификаторов
  - проверка уникальности пользовательских алиасов
  - редирект на оригинальный URL
- Настроил формы (WTForms) и серверный рендеринг (Jinja2)
- Реализовал загрузку файлов через веб-интерфейс
- Настроил миграции базы данных (Flask-Migrate)
- Организовал структуру проекта по слоям (views, models, forms, API)

## Основные возможности

- Сокращение длинных ссылок
- Пользовательские короткие ссылки
- Автоматическая генерация коротких URL
- Редирект по короткой ссылке
- REST API
- Загрузка файлов через интерфейс

## Технологический стек

Backend: Python 3.9+, Flask  
База данных: SQLite (dev)  
ORM: SQLAlchemy  
Формы: WTForms  
Миграции: Flask-Migrate  
Шаблоны: Jinja2  

## Установка и запуск

```bash
git clone <ссылка-на-репозиторий>
cd yacut
```

### Виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux / macOS
venv\Scripts\activate     # Windows
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Переменные окружения

Создайте файл `.env`:

```
FLASK_APP=yacut
FLASK_ENV=development
SECRET_KEY=your_secret_key
DATABASE_URI=sqlite:///db.sqlite3
DISK_TOKEN=your_yandex_disk_token
```

### Миграции и запуск

```bash
flask db upgrade
flask run
```

Приложение будет доступно по адресу:
http://127.0.0.1:5000/

## Примеры API

Создание короткой ссылки  
POST /api/id/

Переход по короткой ссылке  
GET /{short_id}/
